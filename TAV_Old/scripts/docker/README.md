## ğŸ’¾ Backup & Restore

### Quick Start / Local:
```bash
# Backup
cp backend/data/tav_engine.db backup_$(date +%Y%m%d).db

# Restore
cp backup_20251112.db backend/data/tav_engine.db
```

### Docker Production:
```bash

docker cp tav-backend-prod:/app/data/tav_engine.db ./backup_$(date +%Y%m%d).db

# Restore
docker cp backup_20251112.db tav-backend-prod:/app/data/tav_engine.db
docker compose -f docker/docker-compose.production.yml restart backend
```
